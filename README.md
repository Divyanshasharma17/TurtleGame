🐢 Turtle Dodge Game
A simple and fun arcade-style dodge game built using Python’s built-in turtle module.
Control the turtle and dodge falling enemies to achieve the highest score!

🚀 Features
   🎮 Easy controls (Left / Right movement)
   🔴 Random enemy spawning
   ⚡ Increasing difficulty over time
   📊 Live score display
   💀 Game Over screen
🕹️ Controls
   ⬅️ Left Arrow → Move left
   ➡️ Right Arrow → Move right
🧠 How It Works
    Enemies spawn randomly at the top of the screen
    They move downward continuously
    Speed increases as your score grows
    Collision with enemy → Game Over
📦 Tech Stack
    Python 🐍
   Turtle Graphics 🎨
📥 Run the Game by using the command :- python turtle_game.py
